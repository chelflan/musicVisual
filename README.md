# musicVisual

> 用web audio api 操作音频，对其分析，再结合canvas画图，将音频效果以图形的形式展示出来。


###用到的相关技术

- WebAudio
- canvas
- express

**线上显示效果[戳这里](http://musicview-ailidan.rhcloud.com/);**

**关于具体的实现思路和方法已记录在[博客](http://eidolons-ailidan.rhcloud.com/detials?id=9)里。**
